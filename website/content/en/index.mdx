A modular, extensible WeChat iLink Bot SDK — available in **Node.js**, **Go**, and **Rust**. Plus a **Pi coding agent extension** that lets you chat with Pi from WeChat. [GitHub](https://github.com/jiweiyuan/wechatbot).

<Section id="quick-start" title="Quick start">

```bash
npm install @wechatbot/wechatbot
```

```typescript
import { WeChatBot } from '@wechatbot/wechatbot'

const bot = new WeChatBot()
await bot.login()              // Shows QR code → scan in WeChat

bot.onMessage(async (msg) => {
  await bot.sendTyping(msg.userId)
  await bot.reply(msg, `Echo: ${msg.text}`)
})

await bot.start()
```

Scan the QR code in WeChat → your bot is live. Credentials auto-saved, session auto-recovered.

<Section id="pi-agent" title="Pi Agent extension">

Chat with your [Pi coding agent](https://github.com/badlogic/pi-mono) directly from WeChat. One command, one QR scan.

```bash
pi -e /path/to/wechatbot/pi-agent/src/index.ts

# In pi:
/wechat        # Shows QR → scan in WeChat → connected!
```

WeChat messages become Pi prompts. Pi responses are sent back to WeChat. Typing indicator shows while Pi thinks. [Full docs →](/en/pi-agent)

<Section id="protocol" title="Protocol overview">

The iLink Bot API is HTTP/JSON, centered on `context_token` — every reply must echo back the token from the incoming message.

- **`GET /get_bot_qrcode`** — QR code for login
- **`POST /getupdates`** — long-poll messages (35s hold)
- **`POST /sendmessage`** — send text or media
- **`POST /sendtyping`** — typing indicator
- **`POST CDN /upload`** — AES-encrypted media upload
- Error `-14` → session expired → auto re-login

</Section>
