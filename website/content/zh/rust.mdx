Rust SDK 基于 `tokio` + `reqwest` + `serde` 构建，异步、类型安全。所有协议类型派生 `Serialize + Deserialize + Clone + Debug`。

<Section id="install" title="安装">

```toml
[dependencies]
wechatbot = "0.1"
tokio = { version = "1", features = ["full"] }
```

</Section>

<Section id="quick-start" title="快速开始">

```rust
use wechatbot::{WeChatBot, BotOptions};

#[tokio::main]
async fn main() {
    let bot = WeChatBot::new(BotOptions::default());
    let creds = bot.login(false).await.unwrap();
    println!("已登录: {}", creds.account_id);

    bot.on_message(Box::new(|msg| {
        println!("{}: {}", msg.user_id, msg.text);
    })).await;

    bot.run().await.unwrap();
}
```

</Section>

<Section id="configuration" title="配置">

```rust
let bot = WeChatBot::new(BotOptions {
    base_url: None,     // 默认: ilinkai.weixin.qq.com
    cred_path: None,    // 默认: ~/.wechatbot/credentials.json
    on_qr_url: Some(Box::new(|url| println!("扫码: {}", url))),
    on_error: Some(Box::new(|err| eprintln!("错误: {}", err))),
});
```

</Section>

<Section id="messages" title="消息处理">

```rust
bot.on_message(Box::new(|msg| {
    match msg.content_type {
        ContentType::Text  => println!("文本: {}", msg.text),
        ContentType::Image => println!("图片: {} 张", msg.images.len()),
        ContentType::Voice => {
            for v in &msg.voices {
                println!("语音: {:?} ({}ms)", v.text, v.duration_ms.unwrap_or(0));
            }
        }
        ContentType::File  => println!("文件: {:?}", msg.files[0].file_name),
        ContentType::Video => println!("视频"),
    }
})).await;
```

</Section>

<Section id="sending" title="发送消息">

```rust
bot.reply(&msg, "Echo: 你好").await?;   // 回复
bot.send(user_id, "你好").await?;        // 主动发送
bot.send_typing(user_id).await?;         // 显示输入中
```

</Section>

<Section id="errors" title="错误处理">

```rust
match result {
    Err(WeChatBotError::Api { errcode: -14, .. }) => {
        // 会话过期 —— 自动处理
    }
    Err(WeChatBotError::NoContext(uid)) => {
        // 尚无 context_token
    }
    Err(WeChatBotError::Transport(e)) => {
        // 网络错误
    }
    _ => {}
}
```

</Section>

<Section id="structure" title="项目结构">

```
rust/src/
├── lib.rs           ← 公开导出
├── types.rs         ← 所有类型（serde）
├── error.rs         ← 错误层次（thiserror）
├── protocol.rs      ← iLink API 调用（reqwest）
├── crypto.rs        ← AES-128-ECB + 测试
└── bot.rs           ← WeChatBot 客户端
```
