一个 **pi 扩展**，将微信桥接到你的 [Pi 编程助手](https://github.com/badlogic/pi-mono)。微信扫码 → 直接在微信里与 Pi 对话。使用 `@wechatbot/wechatbot` SDK 处理所有 iLink 协议操作。

<Section id="how-it-works" title="工作原理">

<CodeBlock lang="diagram" lineHeight="1.3" showLineNumbers={false}>
{`┌──────────────┐      ┌──────────────┐      ┌──────────┐
│  微信用户     │ ←──→ │  iLink API   │ ←──→ │ Pi 助手  │
│  (手机)      │      │  (腾讯)      │      │ (笔记本) │
└──────────────┘      └──────────────┘      └──────────┘
                              ↑
                   @wechatbot/wechatbot SDK`}
</CodeBlock>

1. 在 pi 中运行 `/wechat`
2. 扩展创建 `WeChatBot` 实例 → 调用 `bot.login()`
3. SDK 调用 iLink API → 获取二维码 → 在 pi TUI 中显示
4. 用手机微信扫码
5. **微信消息 → `pi.sendUserMessage()`** → 成为 pi prompt
6. **`pi.on('agent_end')` → `bot.reply()`** → 发回微信
7. Pi 思考时显示"对方正在输入中"

</Section>

<Section id="install" title="安装">

**方式一：** 直接加载：

```bash
pi -e /path/to/wechatbot/pi-agent/src/index.ts
```

**方式二：** 复制到扩展目录（自动加载）：

```bash
# 全局
cp -r pi-agent/ ~/.pi/agent/extensions/wechat-bridge/

# 项目级
cp -r pi-agent/ .pi/extensions/wechat-bridge/
```

</Section>

<Section id="usage" title="使用方法">

```bash
> /wechat

  ╔══════════════════════════════════════════╗
  ║    📱 在微信中扫描此二维码               ║
  ╚══════════════════════════════════════════╝

  https://weixin.qq.com/x/cAbCdEfGhIj

[wechat] 已扫码 — 在微信中确认
[wechat] 登录确认
✓ 微信已连接！账号: e06c1ceea05e@im.bot
```

现在从微信发送消息 —— Pi 将其作为 prompt 处理，并把回复发回微信。

</Section>

<Section id="message-flow" title="消息流程">

<CodeBlock lang="diagram" lineHeight="1.3" showLineNumbers={false}>
{`微信用户发送 "帮我重构 auth 模块"
         │
         ▼
  iLink API（长轮询 getupdates）
         │
         ▼
  扩展收到 IncomingMessage
         │
         ├── bot.sendTyping(userId)        → "对方正在输入中..."
         │
         ├── pi.sendUserMessage(msg.text)  → 成为 pi prompt
         │
         ▼
  Pi 助手处理（工具、思考...）
         │
         ▼
  pi.on('agent_end') 触发
         │
         ├── 提取助手文本
         ├── bot.stopTyping(userId)
         └── bot.reply(msg, text)          → 发回微信`}
</CodeBlock>

</Section>

<Section id="commands" title="命令">

```
/wechat              开始登录（显示二维码）
/wechat --force      强制重新登录（新二维码）
/wechat-disconnect   断开微信
/wechat-send <文本>  手动发送文本给微信用户
```

</Section>

<Section id="features" title="功能与限制">

**功能：**
- 零配置 —— `/wechat` 加扫码
- 凭证持久化 —— `~/.wechatbot/`，重启跳过扫码
- 会话恢复 —— 收到 `-14` 自动重新登录
- 输入指示器 —— Pi 思考时显示"对方正在输入中"
- 文本分片 —— 长回复在 2000 字符处分片
- 优雅关闭 —— `session_shutdown` 时停止

**限制：**
- 仅文本消息（图片/语音/文件收到但不作为 prompt 处理）
- 同时只支持一个微信用户
- 二维码显示为 URL（非终端渲染的实际二维码图像）
- 需要 pi 在交互模式下运行

</Section>
