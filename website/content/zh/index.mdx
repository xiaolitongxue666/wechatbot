模块化、可扩展的微信 iLink Bot SDK —— 提供 **Node.js**、**Go**、**Rust** 三种语言。同时内含 **Pi 编程助手扩展**，微信扫码即可与 Pi 对话。[GitHub](https://github.com/jiweiyuan/wechatbot)。

<Section id="quick-start" title="快速开始">

```bash
npm install @wechatbot/wechatbot
```

```typescript
import { WeChatBot } from '@wechatbot/wechatbot'

const bot = new WeChatBot()
await bot.login()              // 显示二维码 → 微信扫码

bot.onMessage(async (msg) => {
  await bot.sendTyping(msg.userId)
  await bot.reply(msg, `Echo: ${msg.text}`)
})

await bot.start()
```

在微信中扫描二维码 → 机器人上线。凭证自动保存，会话自动恢复。

</Section>

<Section id="pi-agent" title="Pi Agent 扩展">

直接在微信里与你的 [Pi 编程助手](https://github.com/badlogic/pi-mono) 对话。一条命令，一次扫码。

```bash
pi -e /path/to/wechatbot/pi-agent/src/index.ts

# 在 pi 中输入：
/wechat        # 显示二维码 → 微信扫码 → 连接成功！
```

微信消息变成 Pi prompt。Pi 的回复发回微信。Pi 思考时显示"对方正在输入中"。[完整文档 →](/zh/pi-agent)

</Section>

<Section id="protocol" title="协议概览">

iLink Bot API 基于 HTTP/JSON，核心是 `context_token` —— 每个回复必须回传入站消息中的 token。

- **`GET /get_bot_qrcode`** —— 获取登录二维码
- **`POST /getupdates`** —— 长轮询消息（35 秒挂起）
- **`POST /sendmessage`** —— 发送文本或媒体
- **`POST /sendtyping`** —— 输入状态指示器
- **`POST CDN /upload`** —— AES 加密媒体上传
- 错误码 `-14` → 会话过期 → 自动重新登录

</Section>
